# Tic Tac Toe App

This is a simple Tic Tac Toe game I built while learning Android development.  
The game is for 2 players and runs on Android phones.

## Features

- Two player game (X and O)
- Win detection for rows, columns and diagonals
- Draw condition
- Reset option

## Tech Used

- Java
- Android Studio
- XML Layout

## What I Learned

- Working with ImageView and onClick events
- Handling UI with ConstraintLayout
- Using arrays and conditions for game logic
- Passing data and updating UI

## How to Run

1. Download or clone the project
2. Open it in Android Studio
3. Run on emulator or a physical Android device
